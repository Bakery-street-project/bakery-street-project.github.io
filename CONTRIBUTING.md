# Contributing to bakery-street-project.github.io

Thank you for your interest in contributing to bakery-street-project.github.io!

## Getting Started

1. Fork the repository
2. Clone your fork locally: `git clone https://github.com/bakery-street-project/bakery-street-project.github.io.git`
3. Create a new branch: `git checkout -b feature/your-feature-name`
4. Make your changes
5. Commit your changes: `git commit -m 'Add some feature'`
6. Push to the branch: `git push origin feature/your-feature-name`
7. Open a Pull Request
